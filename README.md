# React Contact Cards SPA

## Overview
A React Single Page Application (SPA) that allows users to add and display contact cards using reusable React components.

## Features
- Add new contacts
- Display contacts as cards
- Reusable React components
- React Hooks (useState)
- Responsive UI
- LocalStorage support (if implemented)

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Vite

## Project Structure
- ContactForm Component
- ContactCard Component
- UserList Component

## How to Run

npm install

npm run dev
